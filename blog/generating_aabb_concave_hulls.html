Generating Axis-Aligned Concave Hulls

Author: Iyad Ahmed

Pseudo-code


Generate 2D outlines of mesh projections on X, Y and Z planes

Extrude each outline infinitely (or just enough so that they all intersect, use the dimensions of axis-aligned bounding box as extrusion lengths)

The concanve hull is the intersection of all extruded parts

Notes:

  • 2D outline of mesh projection should be computable with backface culling (with Z-buffer for "smooth" culling)
  • Hopefully the culling and the boolean intersection can be both done on the GPU
